# LeetCode 620 - Not Boring Movies

## Problem Statement

Table: Cinema

Write a query to report movies with:

• an **odd-numbered id**  
• a **description that is not 'boring'**

Return the result **ordered by rating in descending order**.

Return the result with:

| Column | Description |
|------|-------------|
| id | movie id |
| movie | movie name |
| description | movie description |
| rating | movie rating |

---

## Table Schema

### Cinema

| Column | Type |
|------|------|
| id | NUMBER |
| movie | VARCHAR2(50) |
| description | VARCHAR2(50) |
| rating | NUMBER |

Primary Key

(id)

<br>

## Expected Output

| id | movie | description | rating |
|----|------|-------------|--------|
| 5 | House card | Interesting | 9.1 |
| 1 | War | Great 3D | 8.9 |

---

## Create Table (Oracle)

```sql
CREATE TABLE Cinema (
    id NUMBER PRIMARY KEY,
    movie VARCHAR2(50),
    description VARCHAR2(50),
    rating NUMBER
);
