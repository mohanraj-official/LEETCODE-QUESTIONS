# LeetCode 175 - Combine Two Tables

## Problem Statement

Two tables exist: `Person` and `Address`.

Write an SQL query to report:

- firstName
- lastName
- city
- state

Return all persons from the `Person` table.

If a person does not have an address, return `NULL` for city and state.

---

## Table Schema

### Person

| Column | Type |
|------|------|
| personId | NUMBER |
| firstName | VARCHAR2 |
| lastName | VARCHAR2 |

### Address

| Column | Type |
|------|------|
| addressId | NUMBER |
| personId | NUMBER |
| city | VARCHAR2 |
| state | VARCHAR2 |

---

<BR>

## EXPECTED OUTPUT

| firstName | lastName | city          | state    |
| --------- | -------- | ------------- | -------- |
| Allen     | Wang     | NULL          | NULL     |
| Bob       | Alice    | New York City | New York |

<BR><BR>

## Create Table (Oracle)

```sql
CREATE TABLE Person (
    personId NUMBER PRIMARY KEY,
    firstName VARCHAR2(50),
    lastName VARCHAR2(50)
);

CREATE TABLE Address (
    addressId NUMBER PRIMARY KEY,
    personId NUMBER,
    city VARCHAR2(50),
    state VARCHAR2(50)
);


INSERT INTO Person VALUES (1, 'Allen', 'Wang');
INSERT INTO Person VALUES (2, 'Bob', 'Alice');

INSERT INTO Address VALUES (1, 2, 'New York City', 'New York');
INSERT INTO Address VALUES (2, 3, 'Leetcode', 'California');
