# LeetCode 595 - Big Countries

## Problem Statement

Table: World

A country is considered **big** if:

• population ≥ 25,000,000  
OR  
• area ≥ 3,000,000

Write a query to return the **name, population, and area** of big countries.

Return the result with:

| name | population | area |

---

## Table Schema

### World

| Column | Type |
|------|------|
| name | VARCHAR2 |
| continent | VARCHAR2 |
| area | NUMBER |
| population | NUMBER |
| gdp | NUMBER |

Primary Key

(name)

<br>

## Expected Output

| name | population | area |
|------|-----------|------|
| Afghanistan | 25500100 | 652230 |
| Algeria | 37100000 | 2381741 |

---

```sql
CREATE TABLE World (
