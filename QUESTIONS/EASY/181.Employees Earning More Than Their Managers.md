# LeetCode 181 - Employees Earning More Than Their Managers

## Problem Statement

Table: Employee

Find employees who earn more than their managers.

Write a query to return the **name of employees** whose **salary is greater than their manager's salary**.

Return the result as:

| Column Name |
|-------------|
| Employee |

---

## Table Schema

### Employee

| Column | Type |
|------|------|
| id | NUMBER |
| name | VARCHAR2 |
| salary | NUMBER |
| managerId | NUMBER |

---

## Create Table (Oracle)

```sql
CREATE TABLE Employee (
    id NUMBER PRIMARY KEY,
    name VARCHAR2(50),
    salary NUMBER,
    managerId NUMBER
);
