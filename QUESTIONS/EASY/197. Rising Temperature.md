# LeetCode 197 - Rising Temperature

## Problem Statement

Table: Weather

Write a query to find **all dates where the temperature is higher than the previous day**.

Return the **id** of those records.

---

## Table Schema

### Weather

| Column | Type |
|------|------|
| id | NUMBER |
| recordDate | DATE |
| temperature | NUMBER |

---


## Expected Output

| id |
| -- |
| 2  |
| 4  |


## Create Table (Oracle)

```sql
CREATE TABLE Weather (
    id NUMBER PRIMARY KEY,
    recordDate DATE,
    temperature NUMBER
);
