# LeetCode 1076 - Project Employees II

## Problem Statement

Table: Project

Each row represents an employee working on a project.

Write a query to report **all project_ids that have the maximum number of employees**.

Return the result with:

| Column | Description |
|------|-------------|
| project_id | project id |

---

## Table Schema

### Project

| Column | Type |
|------|------|
| project_id | NUMBER |
| employee_id | NUMBER |

Primary Key

(project_id, employee_id)

<br>

## Expected Output

| project_id |
|------------|
| 1 |

---

## Create Table (Oracle)

```sql

    project_id NUMBER,
    employee_id NUMBER,
