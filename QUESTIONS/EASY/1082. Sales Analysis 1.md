# LeetCode 1082 - Sales Analysis I

## Problem Statement

Table: Sales

Write a query to report the **seller_id of the seller who made the highest total sales price**.

If multiple sellers have the same highest sales, return all of them.

Return the result with:

| Column | Description |
|------|-------------|
| seller_id | seller id |

---

## Table Schema

### Sales

| Column | Type |
|------|------|
| sale_id | NUMBER |
| product_id | NUMBER |
| seller_id | NUMBER |
| buyer_id | NUMBER |
| sale_date | DATE |
| quantity | NUMBER |
| price | NUMBER |

Primary Key

(sale_id)

<br>

## Expected Output

| seller_id |
|-----------|
| 1 |

---

## Create Table (Oracle)

```sql
CREATE TABLE Sales (
    sale_id NUMBER PRIMARY KEY,
    product_id NUMBER,
    seller_id NUMBER,
    buyer_id NUMBER,
    sale_date DATE,
    quantity NUMBER,
    price NUMBER
);
