# LeetCode 1050 - Actors and Directors Who Cooperated At Least Three Times

## Problem Statement

Table: ActorDirector

Each row represents a collaboration between an **actor** and a **director**.

Write a query to find pairs of **actor_id and director_id** where they have worked together **at least 3 times**.

Return the result with:

| Column | Description |
|------|-------------|
| actor_id | actor id |
| director_id | director id |

---

## Table Schema

### ActorDirector

| Column | Type |
|------|------|
| actor_id | NUMBER |
| director_id | NUMBER |
| timestamp | NUMBER |

Primary Key

(actor_id, director_id, timestamp)

<br>

## Expected Output

| actor_id | director_id |
|----------|-------------|
| 1 | 1 |

---
