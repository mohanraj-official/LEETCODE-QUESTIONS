# LeetCode 597 - Friend Requests I: Overall Acceptance Rate

## Problem Statement

Table: FriendRequest  
Table: RequestAccepted

Write a query to find the **overall acceptance rate of friend requests**.

Acceptance Rate =  
(number of accepted requests) ÷ (number of friend requests sent)

Return the result with:

| accept_rate |

Round the result to **2 decimal places**.

If there are no requests, return **0.00**.

---

## Table Schema

### FriendRequest

| Column | Type |
|------|------|
| sender_id | NUMBER |
| send_to_id | NUMBER |
| request_date | DATE |

### RequestAccepted

| Column | Type |
|------|------|
| requester_id | NUMBER |
| accepter_id | NUMBER |
| accept_date | DATE |

<br>

## Expected Output

| accept_rate |
|-------------|
| 0.50 |
