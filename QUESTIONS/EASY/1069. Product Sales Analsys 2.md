# LeetCode 1069 - Product Sales Analysis II

## Problem Statement

Table: Sales

Write a query to report the **total quantity sold for each product**.

Return the result with:

| Column | Description |
|------|-------------|
| product_id | product id |
| total_quantity | total quantity sold |

---

## Table Schema

### Sales

| Column | Type |
|------|------|
| sale_id | NUMBER |
| product_id | NUMBER |
| year | NUMBER |
| quantity | NUMBER |
| price | NUMBER |

Primary Key

(sale_id, year)

<br>

## Expected Output

| product_id | total_quantity |
|------------|---------------|
| 100 | 18 |
| 200 | 5 |

---

## Create Table (Oracle)

```sql
CREATE TABLE Sales (
    sale_id NUMBER,
    product_id NUMBER,
    year NUMBER,
    quantity NUMBER,
    price NUMBER,
    PRIMARY KEY (sale_id, year)
);
