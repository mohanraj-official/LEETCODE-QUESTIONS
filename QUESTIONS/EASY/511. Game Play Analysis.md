# LeetCode 511 - Game Play Analysis I

## Problem Statement

Table: Activity

Write a query to find the **first login date for each player**.

Return the result with:

| Column | Description |
|------|-------------|
| player_id | player id |
| first_login | first login date |

---

## Table Schema

### Activity

| Column | Type |
|------|------|
| player_id | NUMBER |
| device_id | NUMBER |
| event_date | DATE |
| games_played | NUMBER |

Primary Key

(player_id, event_date)

<br>

## Expected Result

| player_id | first_login |
| --------- | ----------- |
| 1         | 2016-03-01  |
| 2         | 2017-06-25  |
| 3         | 2016-03-02  |

<br><br>

## Create Table 

```sql
CREATE TABLE Activity (
    player_id NUMBER,
    device_id NUMBER,
    event_date DATE,
    games_played NUMBER,
    PRIMARY KEY (player_id, event_date)
);
