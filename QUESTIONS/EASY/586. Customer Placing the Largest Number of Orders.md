# LeetCode 586 - Customer Placing the Largest Number of Orders

## Problem Statement

Table: Orders

Write a query to find the **customer who placed the largest number of orders**.

Return the result with:

| customer_number |

It is guaranteed that **exactly one customer** has the largest number of orders.

---

## Table Schema

### Orders

| Column | Type |
|------|------|
| order_number | NUMBER |
| customer_number | NUMBER |

<br>

## Expected Output

| customer_number |
|-----------------|
| 3 |

---

## Create Table (Oracle)

```sql
CREATE TABLE Orders (
    order_number NUMBER PRIMARY KEY,
    customer_number NUMBER
);
