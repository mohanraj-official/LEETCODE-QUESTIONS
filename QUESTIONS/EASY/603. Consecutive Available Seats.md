# LeetCode 603 - Consecutive Available Seats

## Problem Statement

Table: Cinema

Write a query to report all **consecutive available seats** in the cinema.

A seat is available if `free = 1`.

Return the **seat_id** of seats that are part of at least **two consecutive available seats**.

Return the result ordered by `seat_id`.

---

## Table Schema

### Cinema

| Column | Type |
|------|------|
| seat_id | NUMBER |
| free | NUMBER |

Primary Key

(seat_id)

<br>

## Expected Output

| seat_id |
|--------|
| 2 |
| 3 |
| 4 |
| 5 |

---

```sql

    seat_id NUMBER PRIMARY KEY,
    free NUMBER
);
