# LeetCode 607 - Sales Person

## Problem Statement

Table: SalesPerson  
Table: Company  
Table: Orders

Write a query to report the **names of salespersons who did not have any orders related to the company named 'RED'**.

Return the result with:

| name |

---

## Table Schema

### SalesPerson

| Column | Type |
|------|------|
| sales_id | NUMBER |
| name | VARCHAR2 |
| salary | NUMBER |
| commission_rate | NUMBER |
| hire_date | DATE |

### Company

| Column | Type |
|------|------|
| com_id | NUMBER |
| name | VARCHAR2 |
| city | VARCHAR2 |

### Orders

| Column | Type |
|------|------|
| order_id | NUMBER |
| order_date | DATE |
| com_id | NUMBER |
| sales_id | NUMBER |
| amount | NUMBER |

<br>

## Expected Output

| name |
|------|
| Amy |
| Mark |
| Alex |

---

```sql

);

CREATE TABLE Company (

    city VARCHAR2(50)
);

CREATE TABLE Orders (
    order_id NUMBER PRIMARY KEY,
    order_date DATE,
    com_id NUMBER,
    sales_id NUMBER,
    amount NUMBER
);
