# LeetCode 512 - Game Play Analysis II

## Problem Statement

Table: Activity

Write a query to report the **device that each player used for their first login**.

Return the result with:

| Column | Description |
|------|-------------|
| player_id | player id |
| device_id | device used for first login |

---

## Table Schema

### Activity

| Column | Type |
|------|------|
| player_id | NUMBER |
| device_id | NUMBER |
| event_date | DATE |
| games_played | NUMBER |

Primary Key

(player_id, event_date)

<br>

## Expected Output

| player_id | device_id |
| --------- | --------- |
| 1         | 2         |
| 2         | 3         |
| 3         | 1         |

```sql
CREATE TABLE Activity (
    player_id NUMBER,
    device_id NUMBER,
    event_date DATE,
    games_played NUMBER,
    PRIMARY KEY (player_id, event_date)
);
