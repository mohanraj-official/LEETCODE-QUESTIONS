# LeetCode 584 - Find Customer Referee

## Problem Statement

Table: Customer

Write a query to report the **names of customers who are not referred by the customer with id = 2**.

Return the result with:

| name |

---

## Table Schema

### Customer

| Column | Type |
|------|------|
| id | NUMBER |
| name | VARCHAR2 |
| referee_id | NUMBER |

<br>

## Expected Output

| name |
|------|
| Will |
| Jane |
| Bill |
| Zack |

---

## Create Table (Oracle)

```sql
CREATE TABLE Customer (
    id NUMBER PRIMARY KEY,
    name VARCHAR2(50),
    referee_id NUMBER
);
