# LeetCode 627 - Swap Sex of Employees

## Problem Statement

Table: Salary

Write a query to **swap all 'f' and 'm' values** in the `sex` column.

Rules

• Change `'m'` to `'f'`  
• Change `'f'` to `'m'`

Update the table **in a single query** without using a temporary table.

---

## Table Schema

### Salary

| Column | Type |
|------|------|
| id | NUMBER |
| name | VARCHAR2(50) |
| sex | VARCHAR2(1) |
| salary | NUMBER |

Primary Key

(id)

<br>

## Expected Output

After update

| id | name | sex | salary |
|----|------|----|--------|
| 1 | A | f | 2500 |
| 2 | B | m | 1500 |
| 3 | C | f | 5500 |
| 4 | D | m | 500 |

---

## Create Table (Oracle)

```sql
CREATE TABLE Salary (
    id NUMBER PRIMARY KEY,
    name VARCHAR2(50),
    sex VARCHAR2(1),
    salary NUMBER
);

INSERT INTO Salary VALUES (1,'A','m',2500);
INSERT INTO Salary VALUES (2,'B','f',1500);
