# LeetCode 577 - Employee Bonus

## Problem Statement

Table: Employee  
Table: Bonus

Write a query to report the **name and bonus amount of each employee** whose bonus is **less than 1000**.

Return employees who:

• have a bonus less than 1000  
• or do not have a bonus record

Return the result with:

| name | bonus |

---

## Table Schema

### Employee

| Column | Type |
|------|------|
| empId | NUMBER |
| name | VARCHAR2 |
| supervisor | NUMBER |
| salary | NUMBER |

### Bonus

| Column | Type |
|------|------|
| empId | NUMBER |
| bonus | NUMBER |

<br>

## Expected Output

| name | bonus |
|------|------|
| Brad | NULL |
| John | 500 |
| Dan | NULL |

---

## Create Table (Oracle)

```sql
CREATE TABLE Employee (
    empId NUMBER PRIMARY KEY,
    name VARCHAR2(50),
    supervisor NUMBER,
    salary NUMBER
);

CREATE TABLE Bonus (
    empId NUMBER,
    bonus NUMBER
);
