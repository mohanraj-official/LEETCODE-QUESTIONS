# LeetCode 1068 - Product Sales Analysis I

## Problem Statement

Tables: Sales, Product

Write a query to report the **product_name, year, and price** for each sale.

Return the result with:

| Column | Description |
|------|-------------|
| product_name | name of the product |
| year | year of the sale |
| price | price of the product |

---

## Table Schema

### Sales

| Column | Type |
|------|------|
| sale_id | NUMBER |
| product_id | NUMBER |
| year | NUMBER |
| quantity | NUMBER |
| price | NUMBER |

Primary Key

(sale_id, year)

<br>

### Product

| Column | Type |
|------|------|
| product_id | NUMBER |
| product_name | VARCHAR2(50) |

Primary Key

(product_id)

<br>

## Expected Output

| product_name | year | price |
|--------------|------|------|
| Nokia | 2008 | 5000 |
| Apple | 2009 | 6000 |
| Nokia | 2011 | 5500 |

---

## Create Table (Oracle)

```sql
CREATE TABLE Product (
    product_id NUMBER PRIMARY KEY,
    product_name VARCHAR2(50)
);

CREATE TABLE Sales (
    sale_id NUMBER,
    product_id NUMBER,
    year NUMBER,
    quantity NUMBER,
    price NUMBER,
    PRIMARY KEY (sale_id, year)
);
