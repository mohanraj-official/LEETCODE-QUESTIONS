# LeetCode 610 - Triangle Judgement

## Problem Statement

Table: Triangle

Each row contains three integers `x`, `y`, and `z`.

Write a query to check whether these three values can form a **triangle**.

A triangle is valid if:

x + y > z  
x + z > y  
y + z > x

Return the result with:

| x | y | z | triangle |

Where:

• `triangle = 'Yes'` if it forms a triangle  
• `triangle = 'No'` otherwise

---

## Table Schema

### Triangle

| Column | Type |
|------|------|
| x | NUMBER |
| y | NUMBER |
| z | NUMBER |

<br>

## Expected Output

| x | y | z | triangle |
|---|---|---|----------|
| 13 | 15 | 30 | No |
| 10 | 20 | 15 | Yes |

---

## Create Table (Oracle)

```sql
CREATE TABLE Triangle (
    x NUMBER,
    y NUMBER,
    z NUMBER
);
