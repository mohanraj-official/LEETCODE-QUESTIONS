# LeetCode 619 - Biggest Single Number

## Problem Statement

Table: MyNumbers

A **single number** is a number that appears **only once** in the table.

Write a query to find the **largest single number**.

If no single number exists, return **NULL**.

Return the result with:

| num |

---

## Table Schema

### MyNumbers

| Column | Type |
|------|------|
| num | NUMBER |

<br>

## Expected Output

| num |
|-----|
| 8 |
