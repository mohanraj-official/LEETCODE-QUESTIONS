# LeetCode 183 - Customers Who Never Order

## Problem Statement

Table: Customers  
Table: Orders

Find all customers who **never place an order**.

Write a query to return the **names of customers** who do not appear in the `Orders` table.

Return the result in the following format.

| Customers |
|-----------|

---

## Table Schema

### Customers

| Column | Type |
|------|------|
| id | NUMBER |
| name | VARCHAR2 |

### Orders

| Column | Type |
|------|------|
| id | NUMBER |
| customerId | NUMBER |

---

<br><br>

## Expected output

| Customers |
| --------- |
| Henry     |
| Max       |

<br><br>

```sql
CREATE TABLE Customers (
    id NUMBER PRIMARY KEY,
    name VARCHAR2(50)
);

CREATE TABLE Orders (
