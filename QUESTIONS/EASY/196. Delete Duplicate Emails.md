# LeetCode 196 - Delete Duplicate Emails

## Problem Statement

Table: Person

Write an SQL query to **delete all duplicate emails**.

Keep only **one unique email record**.  
Delete the rows with duplicate emails but keep the row with the **smallest id**.

---

## Table Schema

### Person

| Column | Type |
|------|------|
| id | NUMBER |
| email | VARCHAR2 |

---
<br>


## Expected output

### TABLE BEFORE DELETION

| id | email                                       |
| -- | ------------------------------------------- |
| 1  | [john@example.com](mailto:john@example.com) |
| 2  | [bob@example.com](mailto:bob@example.com)   |
| 3  | [john@example.com](mailto:john@example.com) |

<BR>

### TABLE AFTER DELETION

| id | email                                       |
| -- | ------------------------------------------- |
| 1  | [john@example.com](mailto:john@example.com) |
| 2  | [bob@example.com](mailto:bob@example.com)   |


## Create Table (Oracle)

<BR><BR>

```sql
CREATE TABLE Person (
    id NUMBER PRIMARY KEY,
    email VARCHAR2(100)
);

INSERT INTO Person VALUES (1, 'john@example.com');
INSERT INTO Person VALUES (2, 'bob@example.com');
INSERT INTO Person VALUES (3, 'john@example.com');
