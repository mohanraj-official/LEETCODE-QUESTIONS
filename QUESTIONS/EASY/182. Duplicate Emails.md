# LeetCode 182 - Duplicate Emails

## Problem Statement

Table: Person

Write a query to report all the **duplicate emails**.

An email is duplicate if it appears **more than once** in the table.

Return the result in the following format.

| Email |
|------|

---

## Table Schema

### Person

| Column | Type |
|------|------|
| id | NUMBER |
| email | VARCHAR2 |

---
<br>

## Expected Output

| Email                                       |
| ------------------------------------------- |
| [john@example.com](mailto:john@example.com) |


<br><br>

## Create Table (Oracle)

```sql
CREATE TABLE Person (
    id NUMBER PRIMARY KEY,
    email VARCHAR2(100)
);

INSERT INTO Person VALUES (1, 'john@example.com');
INSERT INTO Person VALUES (2, 'bob@example.com');
INSERT INTO Person VALUES (3, 'john@example.com');
