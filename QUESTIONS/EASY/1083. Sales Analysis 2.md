# LeetCode 1083 - Sales Analysis II

## Problem Statement

Tables: Product, Sales

Write a query to report the **buyer_id of buyers who bought S8 but did not buy iPhone**.

Return the result with:

| Column | Description |
|------|-------------|
| buyer_id | buyer id |

---

## Table Schema

### Product

| Column | Type |
|------|------|
| product_id | NUMBER |
| product_name | VARCHAR2(50) |
| unit_price | NUMBER |

Primary Key

(product_id)

<br>

### Sales

| Column | Type |
|------|------|
| sale_id | NUMBER |
| product_id | NUMBER |
| buyer_id | NUMBER |
| sale_date | DATE |
| quantity | NUMBER |
| price | NUMBER |

Primary Key

(sale_id)

<br>

## Expected Output

| buyer_id |
|----------|
| 1 |

---

## Create Table (Oracle)

```sql
CREATE TABLE Product (
    product_id NUMBER PRIMARY KEY,
    product_name VARCHAR2(50),
    unit_price NUMBER
);

CREATE TABLE Sales (
    sale_id NUMBER PRIMARY KEY,
    product_id NUMBER,
    buyer_id NUMBER,
    sale_date DATE,
    quantity NUMBER,
    price NUMBER
);
