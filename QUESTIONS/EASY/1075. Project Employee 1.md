# LeetCode 1075 - Project Employees I

## Problem Statement

Tables: Project, Employee

Write a query to report the **average experience years of employees for each project**.

Return the result with:

| Column | Description |
|------|-------------|
| project_id | project id |
| average_years | average experience of employees |

Round the average to **2 decimal places**.

---

## Table Schema

### Project

| Column | Type |
|------|------|
| project_id | NUMBER |
| employee_id | NUMBER |

Primary Key

(project_id, employee_id)

<br>

### Employee

| Column | Type |
|------|------|
| employee_id | NUMBER |
| name | VARCHAR2(50) |
| experience_years | NUMBER |

Primary Key

(employee_id)

<br>

## Expected Output

| project_id | average_years |
|------------|---------------|
| 1 | 3.50 |
| 2 | 2.00 |

---

## Create Table (Oracle)

```sql
CREATE TABLE Project (
    project_id NUMBER,
    employee_id NUMBER,
    PRIMARY KEY (project_id, employee_id)
);

CREATE TABLE Employee (
    employee_id NUMBER PRIMARY KEY,
    name VARCHAR2(50),
    experience_years NUMBER
);
